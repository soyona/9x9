# Project Brief: 9x9 MathGrid

## 核心目标
- **认知层**：通过具体图形与动效将“相同加数累加”转化为乘法算式，帮助二年级学生理解乘法本质。
- **熟练度**：基于网格矩阵交互，通过梯度控温（难易度）、限时挑战与高质感声光反馈，提升乘法记忆熟练度。

## 核心业务逻辑
1. **第一步（概念理解）**：演示从加法到乘法的演变。生活中具体物品的累加概念。
2. **第二步（练习记忆）**：行列坐标交叉空白处填写对应结果（如 4行5列 填写 20）。

## 设计美学
- 严格遵循 **Apple 官网风格**：高对比度留白、大文本（SF Pro 字体感）、细腻的阶梯式阴影、平滑的贝塞尔曲线过渡，拒绝低俗高饱和度的动画，代之以优雅、克制的精致奖励机制。
